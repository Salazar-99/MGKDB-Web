Welcome to MGKDB

The open access Multiscale Gyrokinetic Database

Open access to simulation data for the fusion research community

MGKDB is a software ecosystem consisting of a Mongo DB database, command-line tools, and a web application for storing, downloading, analyzing, and interacting with gyrokinetic simulation data produced and uploaded by researchers around the world.

Mongo DB

Mongo DB is a non-relational database which allows high levels of flexibility in the structure of stored data. This flexible structure makes the database extensible and uploading data simple. The database is hosted at NERSC and is capable of storage at the terabyte scale.


Command-line Interface

MGKDB provides a command-line interface and a collection of Python scripts for interacting with the database. Users can download, upload, search, and plot quantities of interest. There is also a GUI interface for ease of use.

MGKDB Web

For users that prefer a web-based experience the MGKDB web app provides a way to browse, filter, and download data. Continuing releases will include plotting utilities and an increasing number of filters.

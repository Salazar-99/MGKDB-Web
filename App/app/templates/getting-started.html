Getting started with MGKDB

There are a variety of ways to interact with the MGKDB database.

Web

To get started using the web-based data browser, Sign Up for an account. Shortly after signing up you will recieve an email asking you to verify your account. Once your account is verified, wait for an Admin to approve your account request. You will recieve an additional email upon approval.

Once your account has been approved, you can browse the MGKDB database via the Data page. Here you can search for runs in two collections: Linear or Non-Linear Runs. Once you have selected the collection you would like to search in you can find a run via its database id or search for a subset of runs with the available filters. Each returned run contains a list of the parameters used to create the run as well as ready viewing of any available diagnostic plots. You can also download the entirety of the runs files from the database. Due to the size of the runs, this usually takes a few seconds but in extreme cases could take a few minutes.
